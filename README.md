# 워키토끼 (Workie-Talkie)
socket_io를 사용한 주제에 따른 채팅방을 생성하고, 그 주제에 대한 대화를 할 수 있는 앱

## 프로젝트 기간
- 2024.11.26 ~ 2024.12.09 (2주)

## 팀원 및 담당 파트
### 조정현 (팀장)
- 로그인 기능
- 홈 화면 기능
- 데모 영상 편집
  
### 이의준 (팀원)
- 필터 기능 제작

### 장승리 (팀원)
- 채팅 기능

### 최지희 (팀원)
- 발표 자료 제작 및 발표

## 주요 기능
### 1. 채팅 시스템
- 실시간 1:1 채팅
- 채팅방 관리

### 2. 알림 시스템
- 예약 관련 알림
  - 예약 신청
  - 승인/거절 상태
  - 예약 확정
